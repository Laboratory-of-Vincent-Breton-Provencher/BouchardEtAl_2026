## Overview

Multi-site fiber photometry (GRAB-DA) recordings of dopamine release across the dorsal
striatum (DS), lateral nucleus accumbens (NAc_lat), basolateral amygdala (BLA), and
medial prefrontal cortex (mPFC) — plus additional target regions (TS, NAc_c, OT) — during
Pavlovian conditioning, reward-magnitude, and probabilistic-outcome tasks. These scripts
reproduce all main and supplementary figure panels from the raw/processed recordings.

## Requirements

- Python ≥ 3.9
- Exact package versions used to run these scripts are listed in `requirements.txt`

```bash
pip install -r requirements.txt
```

## Data

Processed data required to run these scripts are available on OSF:
**https://doi.org/10.17605/OSF.IO/DV724**

Download the data and place it in a `Data/` folder at the root of this repository (i.e.,
alongside the `.py` files), so that paths like `Data/av_and_probcond_raster.npy` resolve
correctly. Expected files:

| File | Used by |
|---|---|
| `av_and_probcond_raster.npy` | most scripts (main Pavlovian/probabilistic-outcome dataset) |
| `rewmag_raster.npy` | reward-magnitude scripts |
| `dataLick.csv` | `Figure_S2B.py` |
| `results_multisite_regression2.csv` | generated by the regression scripts — see **Run order** below |

## Run order / script dependencies

Most scripts are self-contained and only need the raw data above. A few depend on the
output of another script rather than raw data directly:

- `Figure_3B-H_S4ACDEFH.py` and `Figure_S4I.py` each run the full alpha/delta regression
  pipeline from scratch and save `results_multisite_regression2.csv`. **Run one of these
  first** if you're starting from a clean checkout.
- `Figure_S4B.py` (k-fold cross-validation) reads `results_multisite_regression2.csv`.
- `Figure_S5EFG.py` reads `results_multisite_regression2.csv` and additionally hardcodes
  the model's alpha/delta parameters for the additional target regions — see the comment
  at the top of that script.

Everything else can be run independently, in any order.

trial-history effect). Only one of these should be Fig 2B in the final manuscript; this
needs to be resolved and one of the two script names/comments updated accordingly.

## Figure output

Scripts display figures via matplotlib; none currently save files automatically. Add
`fig.savefig(...)` calls, or run interactively and export from the plot window, as needed.
